Documentación Introducción

Introducción

Bienvenido a la documentación de DSG PERU TECHNOLOGY. Aquí encontrarás toda la información necesaria para aprovechar al máximo nuestros servicios.

¿Qué ofrecemos?

Nuestros servicios incluyen:

  • Hosting Web: Alojamiento confiable y rápido para tu sitio web
  • Desarrollo Web: Creación de sitios web y aplicaciones personalizadas
  • Gestión de Dominios: Registro y administración de nombres de dominio
  • Certificados SSL: Seguridad y encriptación para tu sitio
  • Soporte 24/7: Asistencia técnica cuando la necesites

Estructura de la Documentación

Esta documentación está organizada en las siguientes secciones:

  • Primeros Pasos: Información básica para comenzar
  • Hosting: Guías detalladas sobre nuestros servicios de hosting
  • API Reference: Documentación técnica de nuestras APIs

Inicio Rápido

Sigue estos pasos para comenzar rápidamente con nuestros servicios:

1. Crear una Cuenta

Regístrate en nuestro panel de control:

1. Visita https://dsgperu.com/register
2. Completa el formulario de registro
3. Verifica tu email
4. Accede a tu panel de control

2. Configurar tu Primer Sitio

Una vez en el panel de control:

  1. Selecciona un plan de hosting
  2. Elige o registra un dominio
  3. Configura tu sitio web
  4. Sube tus archivos

3. Configurar SSL

Activa el certificado SSL gratuito:

Panel de Control > SSL/TLS > Activar SSL Gratuito

Instalación

Guía detallada para instalar y configurar diferentes tipos de aplicaciones en tu hosting.

WordPress

Instalación automática de WordPress:

1. Panel de Control > Aplicaciones
2. Seleccionar WordPress
3. Elegir dominio y directorio
4. Configurar base de datos
5. Completar instalación

Aplicaciones Personalizadas

Para aplicaciones desarrolladas a medida:

# Subir archivos vía FTP
ftp tu-dominio.com
user: tu-usuario
pass: tu-contraseña

# O usar el administrador de archivos
Panel de Control > Administrador de Archivos

Configuración de Hosting

Aprende a configurar y gestionar tu servicio de hosting web.

Panel de Control

Accede a tu panel de control para gestionar todos los aspectos de tu hosting:

URL: https://panel.dsgperu.com
Usuario: tu-email@dominio.com
Contraseña: tu-contraseña-segura

Configuración de Archivos

Estructura recomendada de directorios:

public_html/
├── index.html          # Página principal
├── css/               # Archivos de estilo
├── js/                # Scripts JavaScript
├── images/            # Imágenes del sitio
└── assets/            # Recursos adicionales

Configuración de Base de Datos

Crear y configurar bases de datos MySQL:

1. Panel de Control > Bases de Datos MySQL
2. Crear nueva base de datos
3. Crear usuario de base de datos
4. Asignar permisos al usuario
5. Configurar aplicación con credenciales

Gestión de Dominios

Administra tus dominios y configuraciones DNS.

Registro de Dominios

Pasos para registrar un nuevo dominio:

  1. Verificar disponibilidad del dominio
  2. Seleccionar extensión (.com, .pe, .org, etc.)
  3. Completar información de contacto
  4. Procesar pago
  5. Configurar DNS

Configuración DNS

Registros DNS principales:

Registro A: Apunta el dominio a una dirección IP 
Tipo: A
Nombre: @
Valor: 192.168.1.100
TTL: 3600
Registro CNAME: Crea un alias para el dominio 
Tipo: CNAME
Nombre: www
Valor: tu-dominio.com
TTL: 3600

Transferencia de Dominios

Para transferir un dominio existente:

1. Obtener código de autorización (EPP)
2. Desbloquear dominio en registrador actual
3. Iniciar transferencia en nuestro panel
4. Confirmar transferencia vía email
5. Esperar completación (5-7 días)

Certificados SSL

Configura certificados SSL para asegurar tu sitio web.

SSL Gratuito (Let's Encrypt)

Activación automática de SSL gratuito:

1. Panel de Control > SSL/TLS
2. Seleccionar "SSL Gratuito"
3. Elegir dominio
4. Hacer clic en "Activar"
5. Esperar validación (2-5 minutos)

SSL Premium

Para sitios comerciales que requieren mayor confianza:

  • Validación de Dominio (DV): Verificación básica
  • Validación de Organización (OV): Verificación de empresa
  • Validación Extendida (EV): Máxima confianza

Instalación Manual

Si necesitas instalar un certificado personalizado:

1. Generar CSR (Certificate Signing Request)
2. Enviar CSR a autoridad certificadora
3. Recibir certificado firmado
4. Instalar en panel de control
5. Configurar redirección HTTPS

API - Visión General

Nuestra API REST te permite gestionar tu hosting programáticamente.

URL Base

https://api.dsgperu.com/v1/

Formato de Respuesta

Todas las respuestas están en formato JSON:

{
  "success": true,
  "data": {
    // datos de respuesta
  },
  "message": "Operación exitosa",
  "timestamp": "2024-01-15T10:30:00Z"
}

Códigos de Estado HTTP

  • 200 - Éxito
  • 201 - Creado
  • 400 - Solicitud incorrecta
  • 401 - No autorizado
  • 404 - No encontrado
  • 500 - Error del servidor

Autenticación API

Todas las solicitudes a la API requieren autenticación mediante API Key.

Obtener API Key

Genera tu API Key desde el panel de control:

Panel de Control > Configuración > API Keys > Generar Nueva

Usar API Key

Incluye la API Key en el header de autorización:

Authorization: Bearer tu-api-key-aqui
Content-Type: application/json

Ejemplo de Solicitud

curl -X GET \
  https://api.dsgperu.com/v1/domains \
  -H 'Authorization: Bearer tu-api-key' \
  -H 'Content-Type: application/json'

Endpoints de la API

Referencia completa de todos los endpoints disponibles.

Dominios

GET /domains

Obtiene la lista de todos tus dominios.

{
  "success": true,
  "data": [
    {
      "id": 1,
      "name": "ejemplo.com",
      "status": "active",
      "expires_at": "2025-01-15"
    }
  ]
}
POST /domains

Registra un nuevo dominio.

{
  "domain": "nuevo-dominio.com",
  "years": 1,
  "auto_renew": true
}

Hosting

GET /hosting/accounts

Lista todas las cuentas de hosting.

POST /hosting/accounts

Crea una nueva cuenta de hosting.

{
  "domain": "ejemplo.com",
  "plan": "basic",
  "billing_cycle": "monthly"
}

SSL

POST /ssl/install

Instala un certificado SSL.

{
  "domain": "ejemplo.com",
  "type": "lets_encrypt"
}
